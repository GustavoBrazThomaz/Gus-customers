App. Lista de Funcionários Projeto criado em Angular 13 utilizando Material Angular e API Rest. O projeto tem como intuito mostrar uma lista de funcionários onde a pessoa que estiver utilizando pode ter total controle sobre a lista. sendo assim podendo ver, adicionar, excluir, atualizar e filtrar de acordo com a vontade dela. para começar a utilizar clique em menu e escolha o componente Tabela.
